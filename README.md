# Skeptique

Skeptique is a small tool that reminds you to always **read the fine print** before believing a headline.  
A place to flip the script, catch the *telltales* and *mirages*, and notice the *blind spots* that most people miss.  
Pick a topic, compare perspectives, and look twice.




### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend 
```bash
cd frontend
npm install
npm run dev
```
